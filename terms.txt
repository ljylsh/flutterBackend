본 계약은 귀하와 Google 사이에 체결되는 법적 계약입니다. 귀하는 다음 이용 약관에 동의함으로써 AdWords API에 대한 개발자 토큰을 발급 받는 당사자를 구속할 수 있는 권한이 있음을 보장하게 됩니다(귀하와 해당 당사자는 집합적으로 “귀하”라 칭함). “개발자 토큰”은 본 프로그램에 성공적으로 가입하는 즉시 Google이 귀하에게 제공하는 AdWords API 프로그램 사용자 패스 키입니다. 개발자 토큰과 Google의 독점 AdWords API 및 그 사양에 액세스하고 이를 이용하는 조건으로 귀하는 AdWords API 이용 약관(“AdWords API 계약AdWords 이용 약관(“AdWords 이용 약관”)의 지배를 받습니다. 본 계약에서 “Google”은 귀하와 Adwords 이용 약관을 함께 체결한 Google 법인(예: Google Inc., Google Ireland Limited 등) 및 그 제휴사를 의미합니다.

AdWords API, 모든 개발자 토큰 및 AdWords 사양은 Google의 지적 재산권이자 소유권 정보입니다. AdWords API 사본 및 AdWords API 사양을 사용, 복제 또는 보유하는 모든 권한을 행사하기 위해서는 본 AdWords API 계약을 엄격히 준수해야 합니다. 본 AdWords API 계약의 어떠한 부분이라도 위반하는 경우 (i) 귀하의 AdWords API 액세스 권한은 사전 통보 없이 중단되거나 해지되며 AdWords API를 사용할 수 있는 권한이 소멸되거나 (ii) 이에 대한 손해배상의 예정으로서, 서비스 등급이 하향 조정되고/되거나 사용에 대해 귀하에게 청구될 비용이 변경될 수 있습니다. 본 AdWords API 계약의 전체 또는 일부를 해지하고자 할 경우 귀하는 모든 AdWords API의 사용을 중단해야 합니다.

I. 정의
본 AdWords API 계약에 사용되는 다음 용어들은 다음과 같은 의미를 갖습니다.

AdWords API “사용”의 의미: (A) (i) AdWords API를 통해 Google 서버에 액세스하거나 (ii) AdWords API 클라이언트를 사용하여 AdWords 계정에 정보를 전송하거나 (iii) AdWords API 호출에 대한 응답으로 Google로부터 정보를 수신하기 위해 AdWords API 사양에 기술된 마크업 언어를 사용하는 행위; 그리고/또는 (B) AdWords API 클라이언트를 배포 또는 개발하는 행위.
“계정 소유자”란 계정 기록 소유자 또는 기록 소유자가 해당 계정에 액세스하고 관리할 수 있는 권한을 부여한 당사자를 의미합니다.
“AdWords API 캠페인 관리 데이터”란 생성 기능 및 관리 기능을 사용하여 AdWords API 서버로 전달된 AdWords API 입력 데이터를 의미합니다.
“AdWords API 클라이언트”란 (i) AdWords API 사양을 사용하여 Google 서버에 액세스하거나 통신할 수 있는 소프트웨어(“Adwords API 기본 클라이언트”) 시스템 및 (ii) AdWords API 기본 클라이언트로, 개발자 토큰 사용을 위해 직접적으로 또는 간접적으로 각각 구글의 승인을 받은 AdWords API 데이터와 통신할 수 있는 소프트웨어 시스템을 의미합니다. “통신”이란 통신 메커니즘이나 의미와는 상관없이, 그리고 통신 메커니즘이 이례적으로 별개의 두 소프트웨어 응용 프로그램(예: 파이프 또는 소켓) 사이의 통신 메커니즘으로 간주되는 경우에도 두 소프트웨어 응용 프로그램이 직접 또는 다른 소프트웨어를 통해 단방향 또는 양방향으로 데이터를 전송할 수 있음을 의미합니다.
“AdWords API 데이터”란 AdWords API 보고 데이터와 AdWords API 입력 데이터를 의미합니다.
“AdWords API 입력 데이터”란 AdWords API를 사용하여 Google로 전송된 모든 데이터, 지침 또는 컨텐츠를 의미합니다.
“AdWords API 보고 데이터”란 보고 기능을 사용하여 Google에서 수신된 모든 데이터 또는 컨텐츠를 의미합니다.
“AdWords API 사양”이란 AdWords API 사양과 프로토콜을 명시하거나 그와 관련이 있는, Google이 제공하는 모든 정보 및 문서, 그리고 Google이 제공하는 AdWords API의 구현 및 사용 방법을 의미합니다.
당사자의 “제휴사”란 해당 당사자를 관리하거나 그러한 관리를 받거나 일반적인 관리 하에 있는 법인을 의미합니다. 앞서 언급한 “관리”란 관리법인이 관리받는 법인의 의결권 있는 주식 중 50%를 초과하는 지분을 직•간접적으로 소유하는 것 또는 관리법인이 관리받는 법인의 이사회의 50%를 넘는 구성원을 임명할 수 있는 권리를 의미합니다.
“사용자 지정”이란 Google과 관련이 없는 소프트웨어 응용 프로그램을 만들거나 실행하기 위해 원래 개발된 표준 소프트웨어 API, 모듈, 기능 및 라이브러리(집합적으로 클라이언트의 “사용자 지정 부분” 이라 칭함)를 제외하고, 최종 광고주 전용 AdWords API 클라이언트 또는 내부 전용 AdWords API 클라이언트의 소스 및 객체 코드에 있는 모든 권리(라이센스 권리 포함)를 소유자가 독점적으로 소유하고 유지 관리한다는 것을 의미합니다. “전매”란 (y) 최종 광고주 전용 AdWords API 클라이언트 또는 내부 전용 AdWords API 클라이언트의 사용자 지정 부분에 대한 전체 권리(라이센스 권리 포함)가 새로운 소유자에게 이전되고 (z) 이전 소유자가 자신이 소유한 클라이언트 코드의 사용자 지정 부분에 대한 사본을 폐기하고 더 이상 사용하지 않는 거래를 의미합니다.
“생성 기능”이란 RMF 목록에서 “생성 기능”이라는 제목 아래에 나열된 기능을 의미합니다.
“최종 광고주 전용 AdWords API 클라이언트”란 (a) AdWords API 클라이언트의 단독 사용자 및 (오픈 소스 코드의 소유권이 아닌) 소유자(“소유자”)가 되는 당사자(및 제휴사)만을 위해 개발되고 (b) (타사를 위해 광고를 관리 또는 구매하는 대행사 또는 재판매업자가 아닌) 소유자의 제품 및 서비스를 위한 광고를 관리하는 데에만 사용되는 사용자 지정 AdWords API 클라이언트를 의미합니다.
“내부 전용 AdWords API 클라이언트”란 (a) 단독 소유자가 되는 당사자(및 제휴사)만을 위해 개발되고 (b) (타사를 위해 광고를 관리 또는 구매하는 대행사 또는 재판매업자가 아닌) 소유자의 고객 제품 및 서비스를 위한 광고를 관리하기 위하여 소유자의 피고용인들만이 사용할 수 있는 사용자 지정 AdWords API 클라이언트를 의미합니다. 소유자의 사용자, 파트너 또는 고객이 사용하는 모든 AdWords API 클라이언트는 내부 전용 AdWords API 클라이언트의 정의에서 명시적으로 배제됩니다.
“관리 기능”이란 RMF 목록에서 “관리 기능”이라는 제목 아래에 나열된 기능을 의미합니다.
“보고 기능”이란 RMF 목록에서 “보고 기능”이라는 제목 아래에 나열된 기능을 의미합니다.
“RMF 목록”이란 다음 주소에 나열된 최소 필수 기능을 의미합니다. https://developers.google.com/adwords/api/docs/requirements.
“제 3자”란 Google 또는 귀하 이외의 제 3자를 의미하고 Google 또는 귀하 이외의 제 3자가 소유하거나 제어하고 있는 데이터베이스, 소프트웨어 또는 서비스가 포함되며 이에 국한되지 않습니다.
II. AdWords API 사용
사용 권한. 본 AdWords API 계약의 이용 약관을 준수하는 경우에만 AdWords API를 사용할 수 있습니다. 항상 동일한 개발자 토큰, 그리고 단 하나의 개발자 토큰만 사용해야 합니다. 귀하가 개발하고 Google 에서 승인 받은 AdWords API 클라이언트에서 사용할 목적으로만 개발자 토큰을 신청할 수 있습니다. Google의 서면 동의가 없을 경우, AdWords API 클라이언트에서 어떠한 제 3자 개발자 토큰도 사용할 수 없습니다. 연속 3 개월 동안 사용하지 않을 경우 귀하의 개발자 토큰은 Google 에 의해 종료될 수 있습니다.
계약 위반 AdWords API 클라이언트. 본 AdWords API 계약을 위반하는 AdWords API 클라이언트를 사용해서는 안 됩니다.
AdWords API 보고 데이터.
AdWords API 보고 데이터의 양도. 귀하는 어떠한 제 3자에게도 AdWords API 보고 데이터의 전체 또는 일부를 판매, 재배포, 재라이센스를 하거나 공개 또는 양도할 수 없습니다(특정 계정의 AdWords API 보고 데이터를 해당 계정의 소유자에게 공개하는 경우 제외).
개발자 토큰. 귀하로부터 직, 간접적으로 개발자 토큰을 얻은 경우, 귀하의 개발자 토큰을 사용하는 모든 AdWords API 사용에 대한 책임이 있습니다.
보안. 귀하는 모든 AdWords API 데이터를 일반적으로 허용 가능한 기업 데이터 보안 표준에 따른 안전한 환경이되 최소한 귀하의 기업 데이터에 대한 보안과 동일한 수준의 환경에서 관리하기 위해 항상 적절한 모든 조치를 취해야 합니다. AdWords API를 사용한 모든 데이터 전송은 최소 128비트 SSL 암호화를 통한 보안이 이루어져야 하며 Google과 직접 전송하는 경우에는 최소한 AdWords API 서버에서 허용하는 프로토콜 수준의 보안을 하여야 합니다.
데이터 수집. Google 제품 또는 서비스 기능에서 Google이 명시적으로 허용하는 경우를 제외하고 귀하는 AdWords API 이외의 자동화된 수단(예: 스크래핑 또는 로봇)을 사용하여 Google, AdWords 프로그램 또는 Google이 소유하거나 운영하는 웹사이트 또는 Google 광고를 게재하는 Google 제휴사 사이트로부터 Google 관련 정보를 액세스, 검색 또는 수집(집합적으로 “Google 스크래핑”이라 칭함)할 수 없습니다.
Google 모니터링. 귀하는 품질을 보장하고 Google 제품, 서비스를 그리고 본 이용 약관에 대한 준수를 개선하기 위한 목적으로 Google이 AdWords API 활동을 모니터링할 수 있다는 점에 동의합니다. 귀하는 이러한 모니터링을 방해하거나 Google로부터 AdWords API 활동을 은폐하려고 시도할 수 없습니다. Google은 이러한 방해 행위를 근절하기 위해 모든 기술적 조치를 취할 수 있습니다.
허가된 사용. 귀하는 개발자 토큰 신청서에 명시된 목적을 위해서만 개발자 토큰을 사용할 수 있습니다. 소재 변경이 있다면 변경이 반영되기 최소 2주전AdWords API 클라이언트에 화면 캡처 그리고 (또는) 실물 크기 모형을 다음의 이메일 주소로 보내야 합니다. 3P-Compliance@google.com. 명확히 해두자면, 그 2주일이 지난 후 소재 변경 내용이 반영되기 전 Google의 답변을 기다릴 필요는 없습니다.
III. AdWords API 클라이언트 개발 및 배포
사용 허락. 본 AdWords API 계약의 이용 약관을 준수하는 경우에만 AdWords API 및 AdWords API 사양을 사용하여 AdWords API 클라이언트를 개발하고 배포할 수 있습니다. 예를 들어, 귀하가 사용, 개발 또는 배포하는 모든 AdWords API 클라이언트는 본 계약의 III절에 명시된 기준을 충족해야 합니다.
AdWords API 클라이언트.
계정 데이터. AdWords API 클라이언트는 어떠한 당사자가 Google 스크래핑을 통해 얻은 데이터에 액세스하거나 이를 사용할 수 있도록 허용하지 않습니다. AdWords API 클라이언트는 AdWords API 데이터가 식별 가능하거나 집합적인 형태인지 여부와 상관 없이 어떠한 당사자가 그 당시 최종 사용자(최종 사용자를 대신하고 최종 사용자가 권한을 위임한 당사자 포함)가 아닌 어느 당사자의 계정, 비밀번호, AdWords API 데이터 또는 그 밖의 다른 계정 정보에 액세스하거나 이를 사용하는 것이 가능하도록 하거나 이를 허용해서는 안 됩니다.
정보 전송. 본 AdWords API 계약의 어떠한 상충되는 내용의 조항에도 불구하고AdWords API 클라이언트는 본 API 계약의 III절에 명시적으로 허용된 바에 따라서만 AdWords API 데이터를 사용하거나 전송할 수 있습니다.
AdWords API 데이터의 혼합. 본 III(2)(c)절은 최종 광고주 전용 AdWords API 클라이언트에 적용되지 않습니다.
생성 기능 및 관리 기능에 대한 입력 필드. AdWords API 클라이언트는 AdWords API 캠페인 관리 데이터를 수집하거나 전송하기 위한 입력 필드를, 제3자의 컨텐츠, 혹은 데이터를 수집하거나 제3자에게 전송하기 위한 입력 필드와 동일한 탭이나 스크린에 나타내어서는 안 됩니다. 단, 다음과 같은 경우는 예외로 합니다. 동일한 탭 또는 화면에서 다음과 같은 Google 및 제 3자에 대한 필드는 편집이 허용됩니다. 키워드 검색 유형, 키워드 입찰, 대상 URL, 캠페인 예산, 캠페인 이름, 광고 그룹명, 광고 그룹 입찰.
데이터 복사. AdWords API 클라이언트는 Google과 제 3자 간의 데이터를 복사하는 기능을 제공할 수 없습니다. 만약 귀하가 귀하의 광고 네트워크를 자체적으로 운영하고 있고, AdWords API 클라이언트는 Google 과 귀하의 광고 네트워크 간의 데이터를 복사하는 기능을 제공한다면, 이 모든 것을 최적화하기 위해 일반사용자는 각각의 경우 Adwords 캠페인 또는 해당 네트워크의 캠페인을 수동으로 선택하여야 합니다.
데이터 보고. AdWords API 클라이언트는 AdWords API 보고 데이터를 사용자 인터페이스의 동일한 페이지 또는 제3자 컨텐츠와 함께 별도 탭이나 화면에서 표시할 수 있습니다. 단, AdWords API 클라이언트가 (i) 타인이 해당 데이터를 Google 또는 AdWords 계정으로 간주하도록 만들 수 있는 방식으로 비Google 데이터를 표시하거나 (ii) 해당 데이터를 Google 이외의 당사자 또는 AdWords가 아닌 제품의 것으로 간주하도록 만드는 방식으로 Google 데이터를 표시하거나 (iii) Google 데이터로 구성되는 집계가 어느 정도인지 명확하지 않은 이상 Google 데이터를 집계할 때 비Google 데이터를 표시하여서는 안 된다는 것을 전제합니다. 제 3자의 컨텐츠가 서로 다른 필드에 표시되는 한 AdWords API 보고 데이터를 화면에서 직접 편집할 수 있습니다. AdWords API 클라이언트가 본 III(2)(с)절의 그 밖의 내용을 준수하는 한 AdWords API 클라이언트는 일부 또는 전적으로 제 3자로부터의 데이터 또는 정보를 기반으로 하여 계산한 집계 데이터를 표시할 수 있습니다.
배포 금지. 전매의 경우를 제외하고 최종 광고주 전용 AdWords API 클라이언트 및 내부 전용 AdWords API 클라이언트를 배포(예: 양도, 라이센스 또는 판매)할 수 없습니다. 단, 최종 광고주 전용 AdWords API 클라이언트 또는 내부 전용 AdWords API 클라이언트의 사용자 지정 부분에 대한 모든 권한이 양도되거나 라이센스를 받은 경우, 또는 최종 광고주 전용 AdWords API 클라이언트나 내부 전용 AdWords API 클라이언트가 (i) 소유자(비실시간 환경에서 개발 및 검사하는 동안 임시로 제외) 또는 (ii) 소유자를 대신하여 소유자 자신의 제품 및 서비스를 광고하는 데에만 해당 클라이언트를 사용하는 대리인 이외의 제 3자에 의해 사용되는 경우에는 최종 광고주 전용 AdWords API 클라이언트가 최종 광고주 전용 AdWords API 클라이언트 자격을 중단하고 내부 전용 AdWords API 클라이언트가 내부 전용 AdWords API 클라이언트 자격을 중단한다는 전제를 수반합니다.
데이터 지연. AdWords API 클라이언트는 해당 지연이 24시간을 초과하는 경우 AdWords 시스템에서 사용 가능한 정보와 비교하여 표시된 AdWords 정보가 지연되는 정도를 AdWords API 클라이언트에 명시합니다.
최소 필수 기능. 모든 AdWords API 클라이언트는 최소한 RMF 목록에 명시된 최소 기능만큼 공개해야 합니다. RMF 목록에 특정 기능이 포함되어 경우 해당 기능의 모든 측면과 해당 기능과 관련된 모든 API 호출은 가능해야 하고 공개되어야 합니다. AdWords API 클라이언트는 RMF 목록에 기능이 추가된 후 4개월 이내에 RMF 목록에 추가된 모든 추가 기능을 공개해야 합니다. 본 III(2)(f)절은 최종 광고주 전용 AdWords API 클라이언트 및 내부 전용 AdWords API 클라이언트에 적용되지 않습니다. 단, 사용자가 합리적으로 명확한 위치에 Google의 지출 및 Google을 통한 고객 광고와 관련된 실적(즉, 총 지출, 클릭, 영향)을 공개하는 보고 데이터를 모든 고객에게 기존 보고만큼 자주(최소월 1회 이상) 제공하는 전제를 수반합니다.
보안. 각 AdWords API 클라이언트는 보안을 유지하고 모든 AdWords API 데이터 및 개발자 토큰의 도난을 방지하기 위해 적절한 보호 기능이 있어야 합니다. 이러한 모든 정보는 기업 데이터에 대해 일반적으로 허용 가능한 보안 표준에 따른 안전한 환경 또는 최소한 귀하의 기업 데이터에 대한 보안과 동일한 수준의 환경에서 관리해야 합니다. 모든 AdWords API 클라이언트는 최소한 128비트 SSL 암호화만큼 안전한 프로토콜 또는 Google로 직접 전송하는 경우 최소한 AdWords API 서버에서 허용하는 프로토콜 수준의 보안을 통해 데이터를 전송해야 합니다.
데이터 수집. AdWords API 클라이언트는 Google 스크래핑을 활성화하거나 Google 스크래핑을 통한 데이터를 허용해서는 안 됩니다.
개인 식별 정보. AdWords API 클라이언트는 수집하는 정보 유형 및 해당 정보의 사용 방법을 사용자에게 먼저 알린 후 그러한 사용에 대한 사용자의 명시적인 승낙을 확보하지 않는 한 타인의 개인 식별 정보를 수집할 수 없습니다.
법률 준수. AdWords API 클라이언트는 적용 가능한 모든 정부 법률, 규칙 및 규정은 물론 제 3자의 권리를 준수해야 하며 AdWords API 클라이언트 사용자에 의해 사기성이 있거나 비윤리적이거나 부적절하거나 혼동을 유발할 수 있다고 생각될 수 있는 방식으로 운영될 수 없습니다.
방해 금지 의무. AdWords API 클라이언트는 AdWords API의 정상 작동을 방해하거나 어떠한 방식으로 방해를 시도하지 않아야 합니다. 각 AdWords API 클라이언트는 할당된 개발자 토큰을 AdWords API 사양에 명시된 대로 Google에 전달해야 합니다. Google의 서면 동의를 통해 명시적으로 허용된 경우를 제외하고 AdWords API 클라이언트의 개발자 토큰을 수정할 수 없습니다.
Google 모니터링. AdWords API 클라이언트는 Google의 AdWords API 활동에 대한 Google의 모니터링을 방해하거나 Google이 알지 못하도록 AdWords API 활동을 은폐해서는 안 됩니다. Google은 그와 같은 방해 행위를 근절하기 위해 모든 기술적 조치(AdWords API 클라이언트의 액세스 중지 또는 종료를 포함하지만 이에 국한되지 않음)를 취할 수 있습니다.
자동화된 사용. 귀하는 자동화된 수단을 통해 제 3자에게 AdWords API를 사용할 수 있는 능력을 제공할 수 없습니다. 모든 제 3자 사용은 AdWords API 클라이언트 사용자 인터페이스를 통해 인간 사용자에 의해 이루어져야 합니다.
API 클라이언트 브랜딩.
로고 사용. 본 AdWords API 계약에 따라 필요 시 귀하가 AdWords API 클라이언트에서 본문에 Google이라 지칭할 수 있는 경우를 제외하고 Google이 서면을 통해 동의하지 않는 한 본 AdWords API 계약은 귀하에게 Google 로고 또는 상표를 사용 또는 게재할 어떠한 권리나 Google 브랜드 표시(아래 정의됨)에 대한 어떠한 권리도 허여하지 않습니다.
브랜드 표시 규칙. Google은 아무때나 그 어떤 이유로도 또는 아무런 이유없이 모든 Google 브랜드 표시에 대한 모든 라이센스를 철회할 수 있습니다. Google의 사전 서면 동의 없이는 브랜드 표시를 어떠한 방식으로든지 변경(예: 색상 또는 크기 변경)할 수 없습니다. 브랜드 표시를 “성인 컨텐츠” 또는 불법 컨텐츠에 게재하거나 연관시킬 수 없습니다. 브랜드 표시에 대한 영업권은 Google의 이익이 되며 귀하는 브랜드 표시와 관련하여 아무런 권리도 가지지 않습니다. Google의 브랜드 표시를 통해 귀하가 얻은 모든 권리, 자격 및 이해 관계는 비 철회 조건으로 Google에 양도하고 위임해야 합니다 사용자는 어떠한 경우라도 브랜드 표시 또는 그의 등록에 대한 이의를 제기하거나 타인이 이에 대해 이의를 제기하도록 조장하거나 Google의 브랜드 표시와 혼동을 일으킬 수 있는 상표, 마크 또는 상호를 등록(법에 따라 해당 권리를 포기할 수 없는 경우 제외)하거나 등록을 시도해서는 안 됩니다. “브랜드 표시”란 상호, 상표, 서비스 상표, 로고, 도메인 이름 및 기타 Google 고유의 브랜드 표시(AdWords 로고를 포함하지만 이에 국한되지 않음)를 의미합니다.
IV. AdWords APIs 일반 사항
사용료.
요금 및 단위. 각 AdWords API 작업 유형이 소비하는 할당량 단위 수는 https://developers.google.com/adwords/api/faq#37445 페이지의 Google AdWords API 요금표에 명시되어 있습니다. “할당량 금액”이란 소비하는 할당량 단위당 청구되는 금액을 의미합니다. https://developers.google.com/adwords/api/faq#37445 페이지의 Google AdWords API 요금표에 명시된 모든 할당량 금액을 매달 Google에 납부해야 합니다. 또한, 귀하는 개발자 토큰과 관련한 모든 AdWords API 작업에 대해 책임을 집니다.
가격 변경.
새로운 기능. Google은 새롭게 사용 가능한 AdWords API 작업에 대하여, 할당량 단위당 할당량 금액 및 새 AdWords API 작업에 의해서 소비되는 할당량 단위의 수를 귀하에게 고지하거나 그와 같은 가격 변경 내역을 https://developers.google.com/adwords/api/faq#37445 페이지의 Google 요금표에 게시함으로써 설정할 수 있습니다.
기존 기능. 기존 AdWords API 작업이 소비하는 할당량 단위의 수에 대한 Google 요금표의 변경사항은 귀하에게 고지한 날로부터 15일 후에 적용됩니다. 할당량 단위당 할당량 금액에 대한 변경사항은 귀하에게 고지한 날로부터 최소 30일 후에 적용됩니다.
희망 가격. 할당량 금액에 대한 희망 가격을 제시할 수 있습니다. 보다 자세한 정보는 https://developers.google.com/adwords/api/faq#37445 페이지를 참조하십시오. Google은 언제든지 희망 가격에 대한 귀하의 액세스 권한을 취소하거나 희망가격을 수정 또는 종료할 수 있는 권리를 가집니다.
지급 조건. 귀하는 Google 과 서면을 통해 달리 정하지 않는 이상 (a) 선지급, (b) 인보이스 지급 기한 또는 (c) Google이 AdWords API 지급을 위해 허용하는 기타 방법을 통해 모든 금액을 미국 달러로 지급해야 합니다. 체납하는 경우 매달 1.5%의 이자(또는 법에서 허용하는 최고 이자)가 적용됩니다. 청구액은 세전 금액입니다. 귀하는 (y) 모든 세금과 정부 부과금 및 (z) Google의 연체료 수령 과정에서 발생하는 합리적인 제반 비용 및 변호사 수수료를 지급할 책임이 있습니다. 인보이스 날짜로부터 60일 이내에 이의를 제기하지 않을 경우, 모든 이의 제기를 포기하는 것으로 간주합니다(이는 귀하의 신용 카드 발행자 권한에 영향을 주지 않습니다). 비용 청구는 오직 Google의 할당량 사용 측정에 따라 이루어집니다. 본 계약의 위와 같은 조건들은 Google에게 그 어떠한 당사자의 크레딧도 연장할 의무를 지우지 않습니다.
보장되지 않는 액세스.
실험 제품 및 프로그램. 다음 조항들은 2012년 7월 8일 이후, 완전한 효력이 생깁니다 : 귀하는 AdWords API의 가용성, 그리고 사용 가능한 조건들은 지속적인 실험 하에 있음을 인정하고 동의합니다 . 귀하는 이 AdWords API 계약이 어떠한 금반언 또는 지속적인 AdWords API 액세스가 필요한 다른 권리도 만들지 않음을 인정하고 동의합니다. 그리고 귀하는 계약, 법률, 또는 다른 법적 이론 하에 Google 에 대해 어떠한 요구도 포기하고 주장하지 않음을 동의합니다.
지속되지 않는 액세스 또는 표준. 귀하는 Google이 귀하의 AdWords API에 대한 액세스를 중지 또는 종료하거나 AdWords API 사양, 프로토콜 또는 액세스 방법을 그 어떠한 이유로 또는 아무런 이유 없이 변경할 수 있으며 그러한 결정에 아무런 책임을 지지 않는다는 점에 동의합니다. 데이터 백업을 하는 한편 AdWords API에 액세스하지 않고 계정을 관리하고 업무를 수행하도록 준비하는 것은 항상 전적으로 귀하의 책임입니다.
서비스 수준 없음. Google은 장애 또는 버그 없이 ADWORDS API를 사용할 수 있다고 진술하거나 보증하지 않으며 이러한 주장을 하지 않았음을 밝힙니다.
준수 검사.
사용자 인터페이스 검사. 귀하는 Google이 언제든지 귀하의 AdWords API 클라이언트 사용자 인터페이스를 검사할 수 있다는 데 동의합니다. 이러한 모든 검사는 정규 업무 시간 중에 진행됩니다. 귀하는 Google이 귀하의 AdWords API 클라이언트 인터페이스 검사를 희망한다고 Google로부터 통보를 받은 후 7일 이내에 Google이 귀하의 사업장을 방문하거나 귀하와 Google이 동의한 방법으로 Google이 합리적으로 만족할 때까지 귀하의 AdWords API 클라이언트에 대한 모든 기능을 검사할 수 있도록 허용해야 합니다. Google의 검사는 Google이 AdWords API 데이터가 표시되거나 입력되는 AdWords API 클라이언트의 각 화면을 철저하게 검사하는 단계로만 구성됩니다. 귀하는 그러한 검사가 종료되는 시점에서 모든 해당 화면을 보여주었음을 서명한 인증서를 Google에 제공해야 합니다.
클라이언트 소프트웨어 원칙. Google에 액세스하는 모든 소프트웨어 응용 프로그램의 개발 및 배포(귀하의 AdWords API 클라이언트의 개발 또는 배포를 포함하지만 이에 국한되지 않음)는 항상 http://www.google.com/corporate/software_principles.html 페이지에 명시된 소프트웨어 원칙을 준수해야 합니다.

비독점 조항. 본 AdWords API 계약은 비독점 계약입니다. 귀하는 Google이 본 AdWords API, AdWords API 클라이언트 또는 기타 제품 또는 서비스와 경쟁할 수 있는 제품 또는 서비스를 개발할 수 있다는 점에 동의합니다.
제 3자 배제. 귀하는 항상 AdWords API를 통해 계정에 액세스하는 계정을 소유한 모든 고객에게 해당 고객의 AdWords 캠페인을 귀하의 서비스 및 개발자 토큰에서 분리하고 해당 고객이 자신의 AdWords 계정에 대한 독점적 관리 권한을 다시 확보할 수 있게 해 주는 쉽고 빠른 기능을 고객이 귀하에게 고지한 날로부터 3 영업일 이내에 제공해야 합니다.
면책. 귀하는 귀하의 Google AdWords API 사용 또는 AdWords API 클라이언트의 개발, 작업, 유지관리, 사용 및 컨텐츠와 관련한 행위 또는 누락(제 3자의 소유권 침해를 포함하지만 이에 국한되지 않음)과 관련하여 발생하는 이의 제기, 소송 또는 절차(집합적으로 “이의 제기”라 칭함)로 인해 발생하는 모든 이의 제기, 비용, 손실, 손해, 채무, 판결 및 제반 비용(변호사 및 기타 전문가의 합리적인 수수료 포함)과 관련하여 Google, 대리인, 제휴사 및 라이센스 사용권자를 면책하고, 방어하고, 손해를 입지 않도록 하여야 합니다. Google의 선택에 따라 귀하는 귀하의 면책 대상이 된 청구와 관련하여 방어와 화해에 대한 권한을 가지게 될 것입니다(단, 이 경우 Google은 이후 언제든지 그러한 청구와 관련한 방어와 화해에 대한 권한을 되받을 것을 선택할 수 있으며, 그러한 경우 귀하는 Google의 사전 서면 동의 없이 화해를 할 수 없습니다),.
해지. 귀하가 본 AdWords API 계약의 규정을 준수하지 않는 경우 본 AdWords API 계약에 포함된 모든 라이센스는 사전 통지 없이 자동 해지됩니다. Google은 어떠한 책임도 지지 않고 이유를 막론하고 언제든지 본 계약을 해지하거나 AdWords API 또는 어떠한 일부 또는 기능을 정지할 수 있는 권리를 가집니다. 해지 또는 중단을 통보 받는 경우 귀하는 AdWords API의 사용 또는 AdWords API 클라이언트의 배포 또는 개발을 중단하고 귀하가 소유 또는 관리하는 모든 AdWords API 사양(귀하의 모든 AdWords API 클라이언트 및 서버를 포함하지만 이에 국한되지 않음)을 삭제해야 합니다. I, II(2) - (4), III(2), III(3)(b), IV(2) 및 IV(5) - (16)절 조항은 본 AdWords API 계약이 해지된 후에도 유효하며 귀하는 각 약관을 따라야 합니다. Google AdWords 프로그램이 프로그램 기능을 지원하는 것을 중단한 경우 AdWords API는 작동을 계속하고 더 이상 지원되지 않는 Google AdWords 프로그램에 대한 오류 코드를 반환합니다.
